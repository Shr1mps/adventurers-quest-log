## Installation

It's always easiest to install modules from the in game add-on browser.

To install this module manually:
1.  Inside the Foundry "Configuration and Setup" screen, click "Add-on Modules"
2.  Click "Install Module"
3.  In the "Manifest URL" field, paste the following url:
`https://github.com/Shr1mps/adventurers-quest-log/releases/latest/download/module.json`
4.  Click 'Install' and wait for installation to complete
5.  Don't forget to enable the module in game using the "Manage Module" button

## Feature Overview

AQL provides a robust system for managing quests, tracking objectives, and rewarding players.

*   **Quest Log Window**: Lists all quests divided into `In Progress`, `Completed`, and `Failed` tabs.
*   **Quest Tracker**: An on-screen tracker to keep active quests visible.
*   **Quest Details**: Fully editable quest details with WYSIWYG editors for descriptions and GM notes.
*   **Objectives**: Track specific goals within a quest.
*   **Rewards**: Drag-and-drop item rewards directly from the quest to actor sheets.
*   **Personal Quests**: Player-specific quest logs.
*   **Sub Quests**: Support for branching narratives and complex quest chains.
*   **Search & Sort**: Easily find quests with search and sorting options.
